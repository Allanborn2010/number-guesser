A simple interactive Python game where the player tries to guess a randomly generated number between 1 and 100.

---

## Description

This game demonstrates basic Python concepts such as loops, conditionals, user input, and randomness.  
Players get feedback if their guess is too high or too low until they guess correctly.

---

## Features

- Random number generation between 1 and 100  
- Tracks the number of attempts  
- Gives hints if the guess is too high or too low  
- Handles invalid input gracefully
